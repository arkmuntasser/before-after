Read Me

Include the two (2) js files and the one (1) css file included and then set your images using data-bgi on each .ic-image. Do not set the background-image property inline.

Then include the following html where you want the comparison to go on your page.

<div class="image-compare">
  <!-- Left Image Box -->
  <span class="ic-image ic-left" data-bgi="ruby.jpg"></span>
  <!-- Right Image Box -->
  <span class="ic-image ic-right" data-bgi="yang.jpg"></span>
  <!-- handle -->
  <span class="ic-handle"></span>
</div>
