Make your before and after comparisons pop!

Read Me

Include the two(2) js files and the one(1) css file included and then set your images as a background-image on .ic-left and .ic-right. Do not set the background-image property inline.

Then include the following html where you want the comparison to go on your page.

<div class="image-compare">
  <div class="ic-image ic-left">
    <!-- Left Image Box -->
  </div>
  <div class="ic-image ic-right">
    <!-- Right Image Box -->
  </div>
  <div class="ic-handle">
    <!-- handle -->
  </div>
</div>
